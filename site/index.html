Your money, double-checked by math.

Financy is a fast, local-first personal finance manager built on real double-entry accounting. Your data lives in a single file you own — and the books can never drift out of balance.

Linux · Windows · macOS — free & open source (MIT)

Built like an accountant, feels like a desktop app

📒

Real double-entry

Every transaction is balanced postings that sum to zero. Balances and net worth are derived, never stored — so they can't go inconsistent.

💵

Money done right

Amounts are integer minor units (cents), with formatting for Rp · $ · € · £. No floating-point money, ever.

🏦

Accounts & net worth

Asset & liability cards, a net-worth overview, and a per-account register with running balance.

🧾

Fast transactions

A date-grouped journal with a familiar Income / Expense / Transfer entry form, live filters and search.

💾

Local-first & yours

Each document is a single .financy SQLite file, always auto-saved with ACID writes. No cloud, no save button.

🔒

Safe by design

Atomic writes, append-only schema migrations, and an automatic .bak before any file upgrade.

Get Financy

Grab the build for your platform from the latest release. Builds are currently unsigned, so Windows/macOS may ask you to confirm before the first run.

🐧 Linux

Download the .tar.xz, extract, and run the app.

🪟 Windows

Run the .exe. SmartScreen → More info → Run anyway.

🍎 macOS

Unzip, move to Applications. First launch: right-click → Open.

Download the latest release

Or build from source: git clone the repo and run make run.
