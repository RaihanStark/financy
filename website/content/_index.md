**Financy** is a fast, local-first personal finance manager built on real
double-entry accounting. Your data lives in a single file you own, and the books
can never drift out of balance.

This is the user guide. New here? Start with
**[Getting Started]({{< relref "getting-started" >}})**, then skim
**[Core Concepts]({{< relref "concepts" >}})** to understand the double-entry model
that makes the numbers trustworthy.
